Ao longo dos anos fui refinando meu setup de trabalho. Estas são as ferramentas que fazem parte do meu dia a dia.

## Editor: VS Code

É o padrão da indústria, e por boas razões. Leve, extensível, e com suporte excelente para TypeScript. Minhas extensões essenciais:

- **GitHub Copilot** — assistente de código com IA
- **Prettier** — formatação automática
- **ESLint** — linting
- **GitLens** — histórico de git visual
- **Thunder Client** — alternativa leve ao Postman

## Terminal: Warp

Depois de anos usando o terminal padrão, migrei para o Warp. Autocomplete inteligente, blocos de comandos, e uma UX pensada para desenvolvedores.

## Notas: Obsidian

Meu segundo cérebro. Uso para documentar projetos, anotar ideias, e manter um diário de aprendizado. Os links bidirecionais são um game changer.

## Gerenciamento de tarefas: Notion + Todoist

Notion para projetos maiores e bases de conhecimento. Todoist para tarefas do dia a dia — simples, rápido, funciona.

## Versionamento: Git + GitHub

Óbvio, mas vale mencionar: commits pequenos e frequentes, branches bem nomeadas, PRs com descrição clara.

## Containers: Docker

Para manter ambientes de desenvolvimento consistentes e fazer deploy sem surpresas. Indispensável.

## A regra de ouro

A melhor ferramenta é aquela que você domina. Não adianta ter o setup mais sofisticado se você não conhece seus atalhos e fluxos.
