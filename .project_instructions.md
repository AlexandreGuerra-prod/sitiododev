# Site Pessoal Alexandre Guerra

## Visão Geral
Site pessoal minimalista para Alexandre Guerra — desenvolvedor web, servidor público, educador e criador de conteúdo. Inspirado em taniarascia.com.

## Stack & Arquitetura
- App Router com TypeScript e Tailwind CSS
- Conteúdo em MDX (arquivos locais em /content/)
- Busca client-side com fuse.js
- Dark/light mode com next-themes (default: dark)
- Banco de dados PostgreSQL para formulário de contato
- Email de notificação para guerra7@gmail.com via Abacus notification API

## Paleta de Cores
- Primary: Ciano/azul elétrico (HSL 187 100% 45-50%)
- Avatar portal: Efeito glow ciano animado (oval vertical, sci-fi portal style)
- Dark mode como padrão

## Rotas
- `/` — Home com hero, posts recentes, notas, projetos, CTAs
- `/sobre` — Sobre com avatar grande e bio (MDX)
- `/blog`, `/blog/[slug]`, `/blog/tag/[tag]` — Blog com posts MDX
- `/notas`, `/notas/[slug]` — Notas rápidas (MDX)
- `/projetos` — Cards de projetos (TypeScript data)
- `/videos` — Vídeos próprios + indicados
- `/recursos` — Livros, cursos, ferramentas, artigos
- `/now` — Página "agora" (MDX)
- `/busca` — Busca com fuse.js
- `/contato` — Formulário com DB + email
- `/feed.xml` — RSS
- `/sitemap.xml`, `/robots.txt` — SEO dinâmico

## Decisões Técnicas
- MDX processado via gray-matter + remark + remark-html (sem MDX bundler)
- Avatar portal: clip-path ellipse + CSS keyframes glow animation
- Footer com 3 colunas: Navegação, Projetos externos, Redes sociais
- Notificação email: NOTIF_ID_FORMULRIO_DE_CONTATO para contato form
